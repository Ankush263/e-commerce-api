# E-commerce API (Go lang)
- ### This repository features a straightforward e-commerce API developed in the GO programming language. It supports all CRUD operations and includes JWT authentication and tokenization. Users can set up stores, manage products, sell items, and purchase new products from the store.

# DB Structure
- ## E - commerce API
    - ### User Model
        - id 
        - username
        - email
        - password
        - phone
        - role 
            - enum
                - seller
                - customer
        - created_at
        - updated_at
    - ### Store Model
        - id
        - owner
        - name
        - description
        - store_type
        - store_id
        - created_at
        - updated_at
    - ### Product Model
        - id
        - name
        - description
        - owner
        - store_id
        - price
        - created_at
        - updated_at
    - ### Buy Model
        - id
        - user
        - owner
        - product
        - store_id
        - total_price
        - created_at
        - updated_at
